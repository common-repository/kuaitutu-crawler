基于深度学习技术的AI采集器，能主动搜索文章并智能解析出标题和正文，通过NLP提取关键词进行比对，最后投递到正确的分类中。


== Description ==

= 无需配置 =
通过深度学习算法模型，AI会自动提取网页中包含的文章和标题

= 主动寻找 =
AI会根据您的关键词，自动到搜索引擎和公众号等渠道帮您找合适的文章发布

= 智能发布 =
AI会通过NLP拆分提取文章关键词，选择哪些发布哪些不发布以及发布到哪个分类下

= 不占资源 =
爬虫运行在独立的电脑，不会耗费您宝贵的服务器性能，提取到的文章会通过RestAPI远程发布到您的博客中

= 站群管理 =
支持同时管理多个网站和爬虫任务

= 更多功能 =
* 支持定时发布
* 支持发布文章撤回
* 支持敏感词检测，不发布带敏感词的文章

== Frequently Asked Questions == 

= 采集到文章里的图片如何下载保存到本地？ =
在wordpress中安装图片保存插件，推荐安装 “qqworld”

= 提示无法连接网站？ =
* 请确保网站能正常访问
* 请确保网站是一个wordpress程序
* 确保 “http://你的域名/wp-json” 这个文件能正常访问（某些nginx服务器下由于配置不正确会访问不了）
* 请到wordpress后台，插件管理，安装快兔兔wordpress连接插件（下载地址）、、
* 如果是通过“宝塔”等工具一键安装的，建议关闭伪静态功能
* 尝试关闭防火墙拦截功能
